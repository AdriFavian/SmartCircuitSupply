# SCS - Smart Circuit Supply Dashboard

**A Digital Solution for Wiring Harness Manufacturing Inventory Control.**

## About The Project
SCS (Smart Circuit Supply) is a web-based dashboard designed to address common supply chain challenges in the automotive wiring harness industry, specifically for **Circuit Supply (CS)** divisions.

This project was built to solve specific pain points such as:
* **Machine Downtime:** Real-time monitoring of machine status (Active/Breakdown).
* **Supply Delays:** Automated "Buffer Stock" alerts when inventory drops below safety limits.
* **Manual Tracking:** Digitalizing the flow from Warehouse to Housing production lines.

## Tech Stack
* **Backend:** Python, Django
* **Database:** SQLite (Prototype), Ready for PostgreSQL
* **UI/Admin:** Django-Jazzmin (Modern Dashboard Interface)
* **Logic:** Django ORM for stock automation
